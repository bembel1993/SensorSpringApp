## Sensor Monitor
____
### Used technologies
#### Framework - Spring Boot
#### API - DispatcherServlet, Thymeleaf
#### Server - Tomcat 9.0.62
#### DB - MySQL
#### Connect with DB - Spring JPA
#### Pattern - MVC
#### Visual - CSS, JS, Bootstrap
#### Integrated Development Environment - Intellij IDEA Ultimate
____
### Features Application
#### 1. Registration unique Person
#### 2. Authorization
#### 3. Two role using app with different opportunities
#### 4. Creat, read, update, delete, search function
